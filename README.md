# Blockchain Based Voting System

A full-stack web-based voting system designed to ensure secure, transparent, and tamper-resistant election processes using blockchain technology.

This system was developed to simulate a real-world voting environment, focusing on how frontend interfaces, backend logic, and blockchain infrastructure integrate into a complete working system.

Key Features:

Secure vote casting using Ethereum smart contracts to ensure immutability and transparency
User authentication and validation to prevent duplicate or unauthorized voting
Real-time vote recording and verification through blockchain transactions
Structured database design for managing user data and system records
Responsive user interface for smooth and accessible voting experience

System Architecture:

Frontend: HTML, CSS, JavaScript (user interaction & voting interface)
Backend: Python (Flask) for application logic and API handling
Blockchain: Ethereum (Solidity, Ganache) for secure vote storage
Database: MySQL for user management and system data

Key Responsibilities:

Designed and developed a full-stack system integrating frontend, backend, and blockchain layers
Implemented RESTful APIs to connect user interface with backend services
Designed database schema and handled CRUD operations for user and voting data
Developed and deployed smart contracts for secure vote validation
Performed testing, debugging, and optimization to ensure system reliability and performance

Outcome:

The system successfully demonstrates how blockchain can be integrated into web applications to create secure and transparent processes, while maintaining usability and system efficiency.
